FuBar_AtlasFu (2010)

AtlasFu is an add-on that works with Atlas, adding an Atlas button to the mod FuBar. When AtlasFu is Enabled, Atlas' minimap button is hidden.

For more information about Atlas, see the read me that comes with Atlas, or visit the developers site at.

www.atlasmod.com
